# DataForge

**AI-Powered Data App Factory** · Hackathon Winner, 1st/50+ teams · 2024

Built a governed AI platform generating Databricks apps from natural language
in under 60 seconds. The system uses a 3-agent LLM architecture:

- **SQL Generator** — Translates natural language queries into valid,
  governed SQL via Unity Catalog
- **Self-Healing Agent** — Automatically retries failed queries up to 3
  times with error-aware corrections
- **Insight Generator** — Asynchronously produces business insights from
  query results

## Technical Stack

Python, Databricks, LLMs, SQL, Unity Catalog
